Solomon Shipping and Trading - Dummy MVP Dataset
Period: 2026-05-18 to 2026-05-24

Files included:
1. customers.csv - customer profiles and destination information
2. shipments.csv - core shipment records and current status
3. pickup_schedule.csv - scheduled pickups and assigned staff
4. payments.csv - invoices, amounts paid, and balances due
5. status_history.csv - shipment status changes over time

Suggested use:
- Use these CSV files first for a Streamlit MVP prototype.
- Later, these tables can be migrated into PostgreSQL or Supabase.
